# Minecraft-clone
Это моя новая и первая в жизни игра на С++. А я Яни и мне 13 лет. 

Minecraft-Clone

Минималистичный воксельный движок на C++ и raylib. Плоский мир 4 чанка по 16×16, толщина — 2 блока. Ставь и ломай блоки, прыгай, исследуй газон с подложкой.

🚀 Запуск

 • Скачай MinecraftClone.exe

 • Положи его в папку с библиотеками (dll-файлы из релиза)

 • Запусти. Всё.

🎮 Управление

 • WASD — движение

 • Пробел — прыжок

 • ЛКМ — поставить блок

 • ПКМ — сломать блок

 • 1-8 — выбрать тип блока

🛠 Сборка
Требуется raylib. Собирается одной командой — в коде всего 283 строки, никаких лишних зависимостей.
